> [!important]
> **Please only install the branches listed below.** All other branches on the [sunnypilot GitHub repository](https://github.com/sunnypilot/sunnypilot) that are not listed below are considered experimental development branches. They are considered unstable, unsupported, and should **not** be installed unless you are explicitly instructed to do so by the sunnypilot team for temporary testing purposes, or you have the technical knowledge to test and troubleshoot independently.
