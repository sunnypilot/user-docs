# Quiet Mode

## What It Does

Quiet Mode suppresses non-critical audio alerts while driving. When enabled, sunnypilot will only play sounds for alerts that require immediate driver attention — all routine chimes and informational sounds are silenced.

## How It Works

Quiet Mode filters audio alerts into two categories:

**Always plays (cannot be silenced):**

| Alert | Reason |
|-------|--------|
| Soft warning | Safety-relevant warning requiring attention |
| Immediate warning | Critical safety alert (e.g., driver intervention needed) |
| Distracted prompt | Driver monitoring detected inattention |
| Repeat prompt | Escalated driver attention reminder |

**Silenced when Quiet Mode is on:**

All other audible alerts, including engagement/disengagement chimes, informational tones, and non-critical notifications.

!!! warning "Safety"
    Quiet Mode never suppresses safety-critical alerts. Warnings for driver inattention, imminent collisions, and system faults will always play regardless of this setting.

## How to Enable

**Settings** -> **Toggles** -> **Quiet Mode**
